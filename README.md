# Meetopia Events - Event Management & Booking Platform

## Project Overview
This project is a comprehensive web application designed for managing and booking events. **Meetopia Events** serves as a dual-sided marketplace, connecting event organizers with attendees. The platform handles everything from event discovery and ticket purchasing to complex organizational management through a dedicated administrative interface.

## Technical Implementation
I was responsible for building the full architecture of the application, ensuring a robust connection between the user interface and the data layer.

* **Frontend Framework:** Developed using **React.js** and **Next.js** to provide a fast, SEO-friendly, and seamless user experience.
* **Database & Infrastructure:** Utilized **PostgreSQL** for reliable data storage and **Docker** for containerization, ensuring consistent development and deployment environments.
* **Auth & Security:** Implemented a secure **user authentication system** with dedicated **role-based access control** (Admin, Organizer, and Regular User).
* **Integration:** Focused on the deep integration of frontend components with backend services to handle real-time ticket availability and event status updates.

## Key Features
* **User Experience:** Advanced event discovery with filtering by category (Music, Theater, Networking), location, and price.
* **Booking System:** Instant ticket purchasing with automated email confirmation and account activation.
* **Organizer Panel:** Dedicated dashboard for partners to create, edit, and manage their own events and internship listings.
* **Admin Control:** A powerful Admin Panel to oversee the entire platform, including user management, partner verification, and event moderation.
* **Profile Management:** Personalized user profiles displaying contact info and a history of purchased tickets ("Evenimentele mele").

## Repository Note
**Code Availability:** This repository contains only selected modules and specific code snippets to demonstrate the architectural logic, authentication flow, and UI/UX patterns. The complete project remains private to protect proprietary integration logic and database schemas.

## Technical Process
The development followed a rigorous full-stack approach:
* **Architecture:** Designing the relational database schema in PostgreSQL to handle multi-role users and event entities.
* **Deployment:** Containerizing the application using Docker to streamline the setup of the Next.js environment and database.
* **Validation:** Implementing secure email verification flows and protected routes based on user roles.
